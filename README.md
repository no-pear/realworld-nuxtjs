### 项目启动

+ npm i 安装项目依赖
+ npm run dev 开发模式
+ npm run build 生成模式打包
+ npm run build 打包后运行

### 项目部署

项目采用 GitHub Actions 持续集成部署，工作流文件 `github/workflows/main.yml`

线上地址：`http://nopear.cn:9000/`

GitHub 仓库：`https://github.com/no-pear/realworld-nuxtjs.git`

### 服务器支持

云服务器一台（linux 系统）

服务器安装 node 环境、pm2
